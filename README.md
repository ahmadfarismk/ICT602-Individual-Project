# ElectriBill ⚡

ElectriBill is an Android application designed to calculate electricity bills based on a tiered pricing system. It allows users to enter their electricity usage, select a billing month, apply a rebate, and view their estimated charges. All records are stored securely in the cloud using Firebase Realtime Database.

This project was developed for **ICT602 – Mobile Technology and Development**.

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ahmadfarismk/ICT602-Individual-Project.git
   ```

2. **Open in Android Studio:**
   - Launch Android Studio and select `Open`.
   - Navigate to the cloned directory and select the `IndividualAssignmentProject` folder.

3. **Firebase Setup:**
   - This project uses Firebase. The `google-services.json` file is already configured for the database.
   - Simply allow Gradle to sync all dependencies.

4. **Build and Run:**
   - Click the green **Run** button (`Shift + F10`) to launch the app on an emulator or physical device.
